# X-SIR: Semantic-Invariant Robust Watermarking Replication

## 📖 项目简介
本项目是对论文 **X-SIR: Semantic-Invariant Robust Watermarking for Large Language Models** 的完整复现，实现了 KGW、SIR 和 X-SIR 三种大模型水印算法，并完成了翻译攻击对比验证。
